# wiki.shelepugin.ru

Это моя маленькая база знаний. Здесь я сохраняю небольшие заметки, инструкции
или гайды, к которым периодически нужен доступ.

## Команды

1.  Запуск dev-сервера

    ```sh
    npm run dev
    ```

2.  Форматирование

    ```sh
    npm run format
    ```

3.  Сборка

    ```
    npm run build
    ```
